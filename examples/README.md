# Examples

## How to run

### Run MySQL container

```shell
cd examples
./run_containers.sh
```

### Init MySQL databases/tables

```shell
cd examples
./run_containers.sh
```

### Run application

```shell
./gradlew :examples:spring-data-r2dbc:bootRun

# or

./gradlew :examples:spring-data-jdbc:bootRun
```
